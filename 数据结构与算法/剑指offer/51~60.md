## 51.构建乘积数组
>既然不能用乘法，分析题目，我们可以将乘积拆为两项。即:
C[i] = A[0] * A[1] *...*A[i-1]
D[i] = A[i + 1] *...* A[n-1]
B[i] = C[i] * D[i]
我们先来计算C[i]，使用数学归纳法：
C[0] = 1
C[1] = A[0]
C[2] = A[0] * A[1]
C[3] = A[0] * A[1] * A[2]
...
我们可以得出规律：C[i] = C[i-1] * A[i -1](i >=1)
我们继续用数学归纳法计算D[i]:
D[n - 1] = 1
D[n - 2] = A[n -1]
D[n - 3] = A[n - 1] * A[n - 2]
我们可以得出规律：D[i] = D[i + 1]*A[i + 1](i <= n-2)

```java
import java.util.ArrayList;
public class Solution {
    public int[] multiply(int[] A) {
        if(A==null || A.length<=1) return A;
        int[] B = new int[A.length];
        int[] C = new int[A.length];
        int[] D = new int[A.length];
        
        C[0]=1;
        for(int i=1;i<C.length;i++){
            C[i]=C[i-1]*A[i-1];
        }
        D[D.length-1]=1;
        for(int i=D.length-2;i>=0;i--){
            D[i]=D[i+1]*A[i+1];
        }
        for(int i=0;i<B.length;i++){
            B[i]=C[i]*D[i];
        }
        return B;
    }
}
```

```java
public class Solution {
    public boolean isNumeric(char[] str) {
        String pattern = "^[-+]?\\d*(?:\\.\\d*)?(?:[eE][+\\-]?\\d+)?$";
```

## 54.字符流中第一个不重复的字符
>LinkedMap在于存储数据时保持进入的顺序与被取出的顺序一致，即LinkedMap是按序存取的。

```java
import java.util.LinkedHashMap;
 
import java.util.Iterator;
public class Solution {
    LinkedHashMap<Character,Integer> map = new LinkedHashMap<>();
    public void Insert(char ch)
    {
        if(map.containsKey(ch)) {
            map.put(ch,-1);
        } else {
            map.put(ch, 1);
        }
    }
    public char FirstAppearingOnce()
    {
        Iterator<Character> iterator = map.keySet().iterator();
       while (iterator.hasNext()) {
           char cur = iterator.next();
           if(map.get(cur) == 1) {
               return cur;
           }
       }
       return '#';
    }
}
```

## 55.链表中环的入口结点
>快慢指针法

```java
/*
 public class ListNode {
    int val;
    ListNode next = null;

    ListNode(int val) {
        this.val = val;
    }
}
*/
public class Solution {
    public ListNode EntryNodeOfLoop(ListNode pHead){
        if(pHead==null) return null;
        ListNode fast=pHead;
        ListNode slow=pHead;
        while(slow!=null && fast!=null && fast.next!=null){
            fast=fast.next.next;
            slow=slow.next;
            while(slow==fast){
                ListNode slow2=pHead;
                while(slow!=slow2){
                    slow=slow.next;
                    slow2=slow2.next;
                }
                return slow2;
            }
        }
        return null;
    }
}
```

## 56.删除链表中重复的结点
```java
/*
 public class ListNode {
    int val;
    ListNode next = null;

    ListNode(int val) {
        this.val = val;
    }
}
*/
public class Solution {
    public ListNode deleteDuplication(ListNode pHead){
        if(pHead==null || pHead.next==null) return pHead;
        ListNode dummy = new ListNode(0);
        dummy.next = pHead;
        ListNode prev = dummy;
        ListNode cur = pHead;
        while(cur!=null){
            while(cur.next!=null && cur.val==cur.next.val){
                cur = cur.next;
            }
            if(prev.next!=cur){
                prev.next = cur.next;
                cur = prev.next;
            }else{
                prev = prev.next;
                cur = cur.next;
            }
        }
        return dummy.next;
    }
}
```

## 58.对称的二叉树
>递归

```java

public class TreeNode {
    int val = 0;
    TreeNode left = null;
    TreeNode right = null;

    public TreeNode(int val) {
        this.val = val;

    }

}
*/
public class Solution {
    boolean isSymmetrical(TreeNode pRoot){
        return helpIsSymmetrical(pRoot,pRoot);
    }
    public boolean helpIsSymmetrical(TreeNode t1,TreeNode t2){
        if(t1==null && t2==null) return true;
        if(t1==null || t2==null) return false;
        return (t1.val==t2.val)

    }
}
```

## 59.按之字形顺序打印二叉树
>和下面的把二叉树打印成多行是一样的，只不过多加了一个判断变量用来判断是否反方向打印

```java
import java.util.ArrayList;
import java.util.Queue;
import java.util.LinkedList;
/*
public class TreeNode {
    int val = 0;
    TreeNode left = null;
    TreeNode right = null;

    public TreeNode(int val) {
        this.val = val;

    }

}
*/
public class Solution {
    public ArrayList<ArrayList<Integer> > Print(TreeNode pRoot) {
        ArrayList<ArrayList<Integer>> res = new ArrayList<>();
        if(pRoot==null) return res;
        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(pRoot);
        boolean flag = true;
        while(!queue.isEmpty()){
            int levNum = queue.size();
            ArrayList<Integer> subList = new ArrayList<>();
            for(int i=0;i<levNum;i++){
                TreeNode temp = queue.poll();
                if(flag){
                    subList.add(temp.val);
                }else{
                    subList.add(0,temp.val);
                }
                if(temp.left!=null) queue.offer(temp.left);
                if(temp.right!=null) queue.offer(temp.right);
            }
            flag = !flag;
            res.add(subList);
        }
        return res;
    }

}
```

## 60.把二叉树打印成多行
>广度优先搜索(DFS)

```java
import java.util.ArrayList;
import java.util.Queue;
import java.util.LinkedList;

/*
public class TreeNode {
    int val = 0;
    TreeNode left = null;
    TreeNode right = null;

    public TreeNode(int val) {
        this.val = val;

    }

}
*/
public class Solution {
    ArrayList<ArrayList<Integer> > Print(TreeNode pRoot) {
        ArrayList<ArrayList<Integer>> res = new ArrayList<>();
        if(pRoot == null) return res;
        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(pRoot);
        while(!queue.isEmpty()){
            int levNum = queue.size();
            ArrayList<Integer> subList = new ArrayList<>();
            for(int i=0;i<levNum;i++){
                TreeNode temp = queue.poll();
                subList.add(temp.val);
                if(temp.left!=null) queue.offer(temp.left);
                if(temp.right!=null) queue.offer(temp.right);
            }
            res.add(subList);
        }
        return res;
    }
}
```
